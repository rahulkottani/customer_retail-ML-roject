# Customer Retail Machine Learning Project

## Overview
This project was developed during the **DevTown 5-Day Machine Learning Bootcamp**. The objective is to analyze customer retail data and compare the performance of different Machine Learning classification algorithms.

## Features
- Data preprocessing
- Missing value handling
- Label Encoding
- Data visualization
- Model training
- Model evaluation
- Accuracy comparison

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Machine Learning Models
- Logistic Regression
- Decision Tree Classifier
- K-Nearest Neighbors (KNN)

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix

## Dataset
Customer Retail Dataset

## Results
The project compares the performance of three classification algorithms and visualizes their accuracy using a bar chart.

## 🎯 Learning Outcomes
During this project, I learned:
- Data preprocessing techniques
- Feature encoding
- Training ML models
- Model evaluation
- Data visualization
- Machine Learning workflow
